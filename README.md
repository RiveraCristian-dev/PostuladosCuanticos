# Postulados Cuánticos

Este proyecto académico implementa programacion en Java  para una simulación de los postulados fundamentales de la mecánica cuántica, aplicando conseptos del temario de las materias de química, probabilidad y estadística 

## Características

- El programa ofrece seleccionar una vista teórica a los postulados ademas de mostrar una simulación de cada uno.
  
## Requisitos

- Java 11+
- NetBeans 15
